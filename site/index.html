Setup guide

Simple installation, but with clear steps because the bot needs configuration.

This project is not a one-click app. To run it safely, the Telegram token, admin ID, and target channel need to be configured first.

1

Install dependencies

pip install -r requirements.txt
2

Configure bot settings

Edit the Telegram token, admin ID, and channel ID in the Python files before launching.

3

Fetch new jobs

python3 fetch_jobs.py
4

Start the bot

python3 bot.py
